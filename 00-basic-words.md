# 📘 English to Urdu/Hinglish Reference

## 👤 Pronouns

| English | Urdu    | Hinglish  |
| ------- | ------- | --------- |
| I       | میں     | Main      |
| We      | ہم      | Hum       |
| You     | تم / آپ | Tum / Aap |
| He      | وہ      | Woh       |
| She     | وہ      | Woh       |
| They    | وہ لوگ  | Woh Log   |
| It      | یہ / وہ | Yeh / Woh |

# 🎯 Useful Classroom Examples

| English                       | Hinglish                           |
| ----------------------------- | ---------------------------------- |
| What are you doing?           | Tum kya kar rahe ho?               |
| Where do you live?            | Tum kahan rehte ho?                |
| Why are you late?             | Tum der se kyun aaye?              |
| How are you?                  | Tum kaise ho?                      |
| Who is your teacher?          | Tumhara teacher kaun hai?          |
| Whose book is this?           | Yeh kitab kiski hai?               |
| How many students are there?  | Kitne students hain?               |
| How much money do you have?   | Kitna paisa hai?                   |
| I study English.              | Main English padhta hoon.          |
| She plays cricket.            | Woh cricket khelti hai.            |
| They have completed the work. | Unhone kaam mukammal kar liya hai. |
